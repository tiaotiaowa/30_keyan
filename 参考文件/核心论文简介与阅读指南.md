### 2. DeepSeek-R1 技术报告
📄 **链接**：https://github.com/deepseek-ai/DeepSeek-R1

**论文简介**：
- **发表时间**：2025年1月
- **研究问题**：如何将大模型的推理能力迁移到小模型
- **核心发现**：
  1. **蒸馏 > RL**：蒸馏在推理能力迁移上优于强化学习
  2. **逆向KL蒸馏**：有效避免模式崩溃
  3. **推理链迁移**：可以将CoT推理能力迁移到小模型

**对你的价值**：⭐⭐⭐⭐⭐
- **验证可行性**：证明推理能力蒸馏是有效的
- **技术借鉴**：逆向KL方法可直接使用
- **实验设计**：可参考其消融实验设计

**核心代码框架**：
```python
# DeepSeek-R1的逆向KL蒸馏
def reverse_kl_loss(student_logits, teacher_logits, temperature=1.0):
    """
    逆向KL散度：KL(p_student || p_teacher)
    优点：避免模式崩溃，保持生成多样性
    """
    student_probs = F.softmax(student_logits / temperature, dim=-1)
    teacher_probs = F.softmax(teacher_logits / temperature, dim=-1)

    # 逆向KL
    rkl = torch.sum(student_probs * (torch.log(student_probs) - torch.log(teacher_probs)), dim=-1)
    return rkl.mean()
```

### 3. AdaSPEC: 选择性蒸馏（NeurIPS 2025 Spotlight）
📄 **arXiv链接**：https://arxiv.org/abs/2510.19779
📄 **GitHub代码**：https://github.com/yuezhouhu/adaspec

**论文简介**：
- **发表时间**：2025年
- **会议**：NeurIPS 2025 Spotlight
- **研究问题**：如何提高蒸馏效率
- **核心创新**：
  1. **选择性token蒸馏**：不是蒸馏所有token
  2. **难度评估**：评估每个token的学习难度
  3. **过滤策略**：过滤难以学习的token，聚焦"易学习"部分

**对你的价值**：⭐⭐⭐⭐⭐
- **高效蒸馏**：可以显著提升蒸馏效率
- **空间词汇应用**：可以将选择性策略应用于空间词汇
- **可复用代码**：GitHub有开源实现

**核心思想**：
```
传统蒸馏：蒸馏所有token
AdaSPEC：只蒸馏"可学习的"token

应用到你研究的场景：
- 空间关系词汇（东、西、南、北）→ 高权重，必须蒸馏
- 普通文本 → 低权重，可选蒸馏
- 复杂推理链 → 选择性蒸馏
```

**阅读重点**：
```
1. 第3节：选择性蒸馏方法
2. 第4节：token难度评估
3. 第5节：实验结果与消融
```
