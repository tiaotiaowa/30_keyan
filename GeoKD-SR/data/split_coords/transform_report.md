## 二、转换规则

### 坐标格式
- 格式: `实体名(经度,纬度)`
- 示例: `北京(116.41,39.90)`

### 替换策略
- 按实体名称长度降序替换，避免部分匹配问题
- 直接使用 `entities[].coords` 字段中的坐标

### 字段处理
| 字段 | 处理方式 |
|------|---------|
| question | **增强** - 在实体后添加坐标 |
| answer | 保持不变 |
| reasoning_chain | 保持不变 |
| entities | 保持不变 |
| 其他字段 | 保持不变 |
