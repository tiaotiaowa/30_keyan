## 4. 关键发现

### 4.1 负面影响
1. **总体准确率下降**: 带坐标版本准确率从23.16%降至19.53%（-3.63%）
2. **方向关系受影响最大**: 准确率下降10.27%
3. **Easy和Medium难度下降明显**: 分别下降6.72%和4.80%
4. **BLEU-4全面下降**: 表明生成文本与参考答案的n-gram匹配度降低

### 4.2 正面影响
1. **复合关系准确率提升**: 从3.66%提升至5.69%（+2.03%）
2. **Hard难度准确率提升**: 从4.47%提升至6.87%（+2.40%）
3. **拓扑关系空间F1提升**: 表明坐标信息对拓扑关键词识别有轻微帮助

### 4.3 可能原因分析
1. **坐标信息干扰**: 坐标数据可能增加了输入的复杂性，使模型难以有效利用空间推理能力
2. **训练数据不匹配**: 模型可能在训练时未充分接触带坐标的问题格式
3. **复合任务受益**: 复合关系涉及多种空间信息，坐标可能提供了额外的上下文线索
4. **难度相关效应**: 简单问题中坐标信息增加了不必要的噪声，复杂问题中坐标提供了有用的辅助信息

---

## 5. 建议

1. **训练策略优化**: 如果要使用带坐标的输入，需要在训练阶段进行针对性微调
2. **输入格式简化**: 考虑将坐标信息以更自然的方式融入问题文本
3. **分层应用**: 可考虑在Hard难度的复合关系问题上使用坐标信息，而在Easy/Medium问题上不使用
4. **进一步实验**: 探索不同坐标表示方式（如相对坐标、标准化坐标）的效果
