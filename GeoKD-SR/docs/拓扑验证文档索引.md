## 技术细节

### 数据结构

```json
{
  "metadata": {
    "version": "2.0",
    "total_count": 11800,
    ...
  },
  "validation_report": {...},
  "prompts": [
    {
      "id": "prompt_0001",
      "relation_type": "topological",
      "topology_subtype": "contains",
      "difficulty": "easy",
      ...
    }
  ]
}
```

### 关键字段

- `relation_type`: 关系类型（topological, directional, metric, composite）
- `topology_subtype`: 拓扑子类型（within, contains, adjacent, disjoint, overlap）
- `difficulty`: 难度等级（easy, medium, hard）
