## 开始使用

### 第一步：阅读概述

**必须先阅读**: [00-概述与前置检查.md](./00-概述与前置检查.md)

该文档包含：
- 项目背景和目标
- 问题分类体系（P0/P1/P2）
- 前置检查清单
- 快速开始指南

### 第二步：完成前置检查

在开始任何实验前，请确保：

- [ ] Python 3.10+ 已安装
- [ ] PyTorch 2.0+ 已安装
- [ ] CUDA 11.8+ 可用
- [ ] GPU显存 >=24GB
- [ ] 教师模型（Qwen2.5-7B）已准备
- [ ] 学生模型（Qwen2.5-1.5B）已准备
- [ ] GeoSR-Bench-3000测试集已准备

### 第三步：按阶段执行

按照阶段1→阶段2→阶段3→阶段4的顺序执行实验。

## 问题分类

### P0级问题（阻塞性）

导致实验无法继续进行的严重问题，必须立即解决。

**示例**：环境依赖缺失、模型加载失败、OOM错误

### P1级问题（重要问题）

影响实验结果准确性或效率的问题。

**示例**：评估指标计算异常、数据质量问题、训练不收敛

### P2级问题（优化建议）

不影响功能但可以改进的问题。

**示例**：代码风格、文档完善、性能优化
