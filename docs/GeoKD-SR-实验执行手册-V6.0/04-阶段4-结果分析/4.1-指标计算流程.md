# 4.1 指标计算流程

## 一、核心指标体系

### 1.1 推理准确率（Reasoning Accuracy, RA）

主要评估指标，综合反映模型推理能力：

```
RA = 0.4 × 推理步骤得分 + 0.3 × 答案准确率 + 0.2 × 完整性 + 0.1 × 清晰度
```

#### 1.1.1 推理步骤得分（40%权重）

评估推理链的逻辑正确性和地理知识运用：

| 评分标准 | 描述 |
|---------|------|
| 5分 | 所有推理步骤正确，逻辑严密，地理知识运用准确 |
| 4分 | 关键推理步骤正确，有次要瑕疵 |
| 3分 | 部分推理步骤正确，存在明显错误 |
| 2分 | 推理链严重不完整或错误 |
| 1分 | 无有效推理步骤 |

#### 1.1.2 答案准确率（30%权重）

```python
def calculate_answer_accuracy(predicted_answer, ground_truth):
    """
    计算答案准确率

    参数:
        predicted_answer: 模型预测答案
        ground_truth: 标准答案

    返回:
        accuracy: 0-1之间的准确率分数
    """
    # 完全匹配
    if predicted_answer == ground_truth:
        return 1.0

    # 语义相似度匹配（适用于长答案）
    similarity = semantic_similarity(predicted_answer, ground_truth)

    # 方向/距离数值匹配（适用于数值型答案）
    if is_numerical_answer(ground_truth):
        numerical_score = numerical_similarity(predicted_answer, ground_truth)
        return max(similarity, numerical_score)

    return similarity
```

#### 1.1.3 完整性（20%权重）

评估回答是否涵盖问题要求的所有要素：

```python
def calculate_completeness(response, question_requirements):
    """
    计算回答完整性

    参数:
        response: 模型回答
        question_requirements: 问题要求清单

    返回:
        completeness_score: 0-1之间的完整性分数
    """
    covered_aspects = 0
    total_aspects = len(question_requirements)

    for aspect in question_requirements:
        if aspect_in_response(aspect, response):
            covered_aspects += 1

    return covered_aspects / total_aspects
```

#### 1.1.4 清晰度（10%权重）

评估表达的清晰性和可理解性：

```python
def calculate_clarity(response):
    """
    计算回答清晰度

    评估维度:
    - 句子结构完整性
    - 专业术语使用准确性
    - 逻辑连接词使用
    - 冗余信息控制
    """
    clarity_score = 0.0

    # 句子结构完整性
    clarity_score += check_sentence_structure(response) * 0.3

    # 术语使用准确性
    clarity_score += check_terminology(response) * 0.3

    # 逻辑连接
    clarity_score += check_logical_flow(response) * 0.2

    # 冗余控制
    clarity_score += check_redundancy(response) * 0.2

    return min(clarity_score, 1.0)
```

## 二、地理特异性指标

### 2.1 方向准确率（八方位）

评估方向推理的准确性，使用八方位体系：

```
N, NE, E, SE, S, SW, W, NW
```

```python
import numpy as np

def calculate_direction_accuracy(predictions, ground_truths, tolerance=45):
    """
    计算方向准确率

    参数:
        predictions: 预测方向列表（角度制，0-360）
        ground_truths: 真实方向列表
        tolerance: 容差角度（默认45度，即相邻方位）

    返回:
        accuracy: 准确率
        confusion_matrix: 8x8混淆矩阵
    """
    # 将角度转换为方位索引（0-7）
    def angle_to_bearing(angle):
        return int(((angle + 22.5) % 360) / 45)

    # 初始化混淆矩阵
    confusion_matrix = np.zeros((8, 8))

    correct = 0
    for pred, true in zip(predictions, ground_truths):
        pred_bearing = angle_to_bearing(pred)
        true_bearing = angle_to_bearing(true)

        confusion_matrix[true_bearing, pred_bearing] += 1

        # 计算方向差
        diff = abs(pred - true) % 360
        if diff > 180:
            diff = 360 - diff

        if diff <= tolerance:
            correct += 1

    accuracy = correct / len(predictions)
    return accuracy, confusion_matrix

# 方位标签
BEARING_LABELS = ['N', 'NE', 'E', 'SE', 'S', 'SW', 'W', 'NW']
```

### 2.2 拓扑混淆矩阵

评估空间拓扑关系的理解准确性：

```python
def calculate_topology_confusion(predictions, ground_truths):
    """
    计算拓扑关系混淆矩阵

    拓扑关系类型:
    - 邻接 (adjacent)
    - 包含 (contains)
    - 相交 (intersects)
    - 分离 (disjoint)
    """
    relation_types = ['adjacent', 'contains', 'intersects', 'disjoint']
    n = len(relation_types)
    confusion_matrix = np.zeros((n, n))

    for pred, true in zip(predictions, ground_truths):
        pred_idx = relation_types.index(pred)
        true_idx = relation_types.index(true)
        confusion_matrix[true_idx, pred_idx] += 1

    return confusion_matrix, relation_types
```

### 2.3 距离MAPE（平均绝对百分比误差）

```python
def calculate_distance_mape(predictions, ground_truths):
    """
    计算距离预测的MAPE

    参数:
        predictions: 预测距离列表（千米）
        ground_truths: 真实距离列表

    返回:
        mape: 平均绝对百分比误差
        errors: 各样本的百分比误差列表
    """
    errors = []
    for pred, true in zip(predictions, ground_truths):
        if true == 0:
            continue  # 避免除零
        ape = abs((pred - true) / true) * 100
        errors.append(ape)

    mape = np.mean(errors)
    return mape, errors
```

### 2.4 推理链完整度

```python
def calculate_reasoning_chain_completeness(response, required_steps):
    """
    计算推理链完整度

    参数:
        response: 模型回答
        required_steps: 必需推理步骤列表

    返回:
        completeness: 完整度分数 (0-1)
        missing_steps: 缺失步骤列表
    """
    detected_steps = []

    for step in required_steps:
        if step_pattern_match(step, response):
            detected_steps.append(step)

    completeness = len(detected_steps) / len(required_steps)
    missing_steps = [s for s in required_steps if s not in detected_steps]

    return completeness, missing_steps

def step_pattern_match(step_type, response):
    """
    检测特定推理步骤

    步骤类型:
    - 'location_identification': 位置识别
    - 'spatial_relation': 空间关系分析
    - 'direction_calculation': 方向计算
    - 'distance_estimation': 距离估算
    - 'topology_inference': 拓扑推理
    """
    patterns = {
        'location_identification': ['位于', '坐标', '经纬度', '位置'],
        'spatial_relation': ['相对于', '之间', '毗邻', '相邻'],
        'direction_calculation': ['方向', '向东', '向西', '向北', '向南'],
        'distance_estimation': ['距离', '约', '千米', '公里'],
        'topology_inference': ['包含', '相交', '边界', '内部']
    }

    keywords = patterns.get(step_type, [])
    return any(keyword in response for keyword in keywords)
```

## 三、指标计算脚本使用方法

### 3.1 计算所有指标

```python
import pandas as pd
from pathlib import Path

def calculate_all_metrics(results_file, output_dir):
    """
    计算所有指标并保存结果

    参数:
        results_file: 实验结果CSV文件路径
        output_dir: 输出目录
    """
    # 读取结果
    df = pd.read_csv(results_file)

    metrics = {
        'ra': [],
        'direction_accuracy': [],
        'distance_mape': [],
        'reasoning_completeness': []
    }

    # 遍历每个样本计算指标
    for _, row in df.iterrows():
        # 计算推理准确率
        ra = calculate_ra(row)
        metrics['ra'].append(ra)

        # 计算方向准确率（如果适用）
        if 'direction' in row:
            dir_acc = calculate_direction_accuracy_single(row)
            metrics['direction_accuracy'].append(dir_acc)

        # 计算距离MAPE（如果适用）
        if 'distance' in row:
            dist_mape = calculate_distance_mape_single(row)
            metrics['distance_mape'].append(dist_mape)

        # 计算推理链完整度
        completeness, _ = calculate_reasoning_chain_completeness(
            row['response'],
            row['required_steps']
        )
        metrics['reasoning_completeness'].append(completeness)

    # 汇总结果
    summary = {
        'RA_mean': np.mean(metrics['ra']),
        'RA_std': np.std(metrics['ra']),
        'Direction_Accuracy': np.mean(metrics['direction_accuracy']),
        'Distance_MAPE': np.mean(metrics['distance_mape']),
        'Reasoning_Completeness': np.mean(metrics['reasoning_completeness'])
    }

    # 保存结果
    output_path = Path(output_dir) / 'metrics_summary.json'
    import json
    with open(output_path, 'w', encoding='utf-8') as f:
        json.dump(summary, f, ensure_ascii=False, indent=2)

    return summary, metrics
```

### 3.2 批量计算

```python
def batch_calculate_metrics(experiment_dirs):
    """
    批量计算多个实验的指标

    参数:
        experiment_dirs: 实验目录列表

    返回:
        all_metrics: 所有实验的指标汇总
    """
    all_metrics = {}

    for exp_dir in experiment_dirs:
        exp_name = Path(exp_dir).name
        results_file = Path(exp_dir) / 'results.csv'

        if results_file.exists():
            summary, metrics = calculate_all_metrics(results_file, exp_dir)
            all_metrics[exp_name] = summary

    # 保存汇总结果
    summary_df = pd.DataFrame(all_metrics).T
    summary_df.to_csv('all_metrics_summary.csv')

    return all_metrics
```

## 五、注意事项

1. **样本量要求**: 每个指标计算至少需要30个样本以确保统计可靠性

2. **异常值处理**: 使用IQR方法识别和处理异常值

3. **缺失值处理**:
   - 数值型指标：使用中位数填充
   - 类别型指标：使用众数填充

4. **多人评估一致性**: 对于主观指标（如推理步骤得分），使用Kappa系数评估评估者间一致性
