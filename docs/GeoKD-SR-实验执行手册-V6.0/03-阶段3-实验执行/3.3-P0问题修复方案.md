## 二、D3-5修复：权重预设矛盾

### 2.1 问题描述

| 属性 | 内容 |
|------|------|
| **编号** | D3-5 |
| **级别** | P0（必须修复） |
| **位置** | Exp9组件权重配置 |
| **问题描述** | Exp9使用预设权重[0.3, 0.25, ...]，但消融实验的目的是确定权重，两者存在逻辑矛盾 |
| **影响分析** | - 权重来源不明，审稿人可能质疑<br>- 与消融实验逻辑冲突<br>- 可复现性不足 |
| **修复方案** | 明确权重为消融实验后的初始配置，建立权重确定流程 |

---

### 2.2 修复方案

#### 方案A：权重确定流程（推荐）

**权重确定流程**:
```
1. 基于消融实验确定组件有效性
   ├─ Exp3b vs Exp2 → C1有效性 → ΔAcc1
   ├─ Exp4 vs Exp2 → C2有效性 → ΔAcc2
   ├─ Exp5 vs Exp2 → C3有效性 → ΔAcc3
   ├─ Exp6 vs Exp2 → C4有效性 → ΔAcc4
   ├─ Exp7 vs Exp2 → C5有效性 → ΔAcc5
   └─ Exp8 vs Exp2 → C6有效性 → ΔAcc6

2. 计算权重比例
   total_acc_gain = Σ ΔAcc_i
   weight_i = ΔAcc_i / total_acc_gain

3. 调整和归一化
   ├─ 硬标签保留基础权重 (0.3)
   ├─ 组件权重总和 = 0.7
   └─ 最终权重 = [0.3, 0.7×weight_1, ..., 0.7×weight_6]

4. 微调优化（可选）
   ├─ 网格搜索：±10%范围
   └─ 选择验证集最优配置
```

**示例计算**:
```
假设消融实验结果：
- ΔAcc1 (C1) = 2.5%
- ΔAcc2 (C2) = 1.8%
- ΔAcc3 (C3) = 1.2%
- ΔAcc4 (C4) = 0.8%
- ΔAcc5 (C5) = 2.0%
- ΔAcc6 (C6) = 1.5%

total_acc_gain = 9.8%

权重计算：
- w_hard = 0.3 (固定)
- w_C1 = 0.7 × (2.5/9.8) ≈ 0.18
- w_C2 = 0.7 × (1.8/9.8) ≈ 0.13
- w_C3 = 0.7 × (1.2/9.8) ≈ 0.09
- w_C4 = 0.7 × (0.8/9.8) ≈ 0.06
- w_C5 = 0.7 × (2.0/9.8) ≈ 0.14
- w_C6 = 0.7 × (1.5/9.8) ≈ 0.11

最终权重: [0.30, 0.18, 0.13, 0.09, 0.06, 0.14, 0.11]
```

## 三、D2-6修复：Holm-Bonferroni校正组数

### 3.1 问题描述

| 属性 | 内容 |
|------|------|
| **编号** | D2-6 |
| **级别** | P0（必须修复） |
| **位置** | 统计显著性检验 |
| **问题描述** | 多重比较校正的组数不明确，影响统计检验的正确性 |
| **影响分析** | - 组数错误导致校正不足或过度<br>- 影响统计结论的有效性<br>- 审稿人可能质疑统计方法 |
| **修复方案** | 明确Holm-Bonferroni校正的组数为11组 |

---

### 3.2 修复方案

**比较组数**: 11组

**组别定义**:
```
统计检验比较组（11组）：
├── 基线组 (2组)
│   ├── Exp1: Direct-SFT
│   └── Exp2: Standard-KD
│
├── 消融组 (6组)
│   ├── Exp3a: C1-Uniform-Weight
│   ├── Exp3b: C1-Adaptive-Weight
│   ├── Exp4: C2-Spatial-CoT
│   ├── Exp5: C3-Reverse-KL
│   ├── Exp6: C4-Self-Distill
│   ├── Exp7: C5-Attention
│   └── Exp8: C6-Progressive
│
└── 完整方法组 (1组)
    └── Exp9: Full-GeoKD-SR

总计: 2 + 6 + 1 = 9个实验组
    + 2个主要对照（Exp1 vs Exp2, Exp9 vs Exp2）
    = 11组比较
```

**主要比较**:
1. Exp2 vs Exp1：验证蒸馏有效性
2. Exp3b vs Exp2：验证C1自适应权重
3. Exp4 vs Exp2：验证C2空间CoT
4. Exp5 vs Exp2：验证C3反向KL
5. Exp6 vs Exp2：验证C4自蒸馏
6. Exp7 vs Exp2：验证C5注意力蒸馏
7. Exp8 vs Exp2：验证C6渐进式蒸馏
8. Exp9 vs Exp2：验证完整方法
9. Exp9 vs Exp3b：完整方法 vs 最佳单组件
10. Exp9 vs max(Exp3-8)：完整方法 vs 所有单组件
11. Exp9 vs Exp1：完整方法 vs 直接微调

### 3.3 Holm-Bonferroni校正流程

```python
# statistical_correction.py - Holm-Bonferroni校正实现
import numpy as np
from scipy import stats

def holm_bonferroni_correction(p_values, alpha=0.05):
    """
    Holm-Bonferroni校正

    Args:
        p_values: p值列表，按从大到小排序
        alpha: 显著性水平（默认0.05）

    Returns:
        拒绝原假设的索引列表
    """
    n = len(p_values)
    rejected = []

    for i, p in enumerate(p_values):
        threshold = alpha / (n - i)
        if p < threshold:
            rejected.append(i)
        else:
            break

    return rejected

# 应用示例
comparisons = [
    ("Exp2 vs Exp1", 0.001),
    ("Exp3b vs Exp2", 0.015),
    ("Exp4 vs Exp2", 0.023),
    ("Exp5 vs Exp2", 0.031),
    ("Exp6 vs Exp2", 0.045),
    ("Exp7 vs Exp2", 0.012),
    ("Exp8 vs Exp2", 0.028),
    ("Exp9 vs Exp2", 0.002),
    ("Exp9 vs Exp3b", 0.008),
    ("Exp9 vs max(Exp3-8)", 0.005),
    ("Exp9 vs Exp1", 0.0005),
]

# 排序p值（从大到小）
sorted_p = sorted([p for _, p in comparisons], reverse=True)

# 应用Holm-Bonferroni校正
rejected_indices = holm_bonferroni_correction(sorted_p)

print(f"总比较数: {len(comparisons)}")
print(f"显著比较数: {len(rejected_indices)}")
print(f"校正后显著性水平: {0.05/len(comparisons):.4f}")
```
