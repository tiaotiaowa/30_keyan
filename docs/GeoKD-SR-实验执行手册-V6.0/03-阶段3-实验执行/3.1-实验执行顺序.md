# 3.1 实验执行顺序

> **文档版本**: V6.0
> **创建日期**: 2026年3月4日
> **目的**: 明确GeoKD-SR实验的执行顺序、依赖关系和具体命令

---

## 一、执行原则

### 1.1 核心原则

1. **基线优先**: 必须先完成基线实验（Exp1-Exp2），建立性能基准
2. **单组件消融**: 在完整方法之前，验证每个组件的独立贡献
3. **数据一致性**: 所有实验使用相同的训练数据和测试集
4. **可复现性**: 每个实验运行3次，使用不同随机种子（42, 43, 44）

### 1.2 实验编号规则

| 编号 | 实验名称 | 类型 | 预计耗时 |
|------|----------|------|----------|
| Exp1 | B1-Direct-SFT | 基线 | 2h |
| Exp2 | B2-Standard-KD | 基线 | 2h |
| Exp3a | C1-Uniform-Weight | C1消融 | 2h |
| Exp3b | C1-Adaptive-Weight | C1消融 | 2h |
| Exp4 | C2-Spatial-CoT | 单组件 | 2h |
| Exp5 | C3-Reverse-KL | 单组件 | 2h |
| Exp6 | C4-Self-Distill | 单组件 | 2h |
| Exp7 | C5-Attention | 单组件 | 2.5h |
| Exp8 | C6-Progressive | 单组件 | 2.5h |
| Exp9 | Full-GeoKD-SR | 完整方法 | 3h |

## 二、阶段A：基线实验（必须先完成）

### 2.1 Exp1: B1-Direct-SFT（对照组）

**目的**: 建立直接监督微调的性能基准，验证蒸馏的有效性

**配置**:
- 模型: Qwen2.5-1.5B + LoRA
- 训练数据: GeoSR-Train (5,000条)
- 损失函数: 仅CrossEntropy(student_logits, labels)
- 超参数: lr=2e-5, batch_size=16, epochs=3, seed=42

**执行命令**:
```bash
cd D:/30_keyan/GeoKD-SR

# 设置环境变量
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=D:/30_keyan/GeoKD-SR:$PYTHONPATH

# 运行Exp1
python experiments/train.py \
    --experiment_name Exp1_Direct_SFT \
    --model_name Qwen/Qwen2.5-1.5B \
    --train_data data/GeoSR-Train.json \
    --test_data data/GeoSR-Bench.json \
    --loss_fn cross_entropy \
    --learning_rate 2e-5 \
    --batch_size 16 \
    --num_epochs 3 \
    --seed 42 \
    --output_dir outputs/Exp1_Direct_SFT \
    --logging_steps 50 \
    --eval_steps 500
```

**预期结果**:
- 准确率: ~65-70%
- 训练时间: ~2小时
- 输出文件: `outputs/Exp1_Direct_SFT/best_model/`

---

### 2.2 Exp2: B2-Standard-KD（通用蒸馏基线）

**目的**: 建立Hinton 2015经典KL蒸馏的性能基准

**配置**:
- 教师模型: Qwen2.5-7B-4bit
- 学生模型: Qwen2.5-1.5B + LoRA
- 损失函数: L_KD = α × L_soft + (1-α) × L_hard
  - α = 0.5
  - 温度 T = 2.0
- 其他参数与Exp1完全一致

**执行命令**:
```bash
cd D:/30_keyan/GeoKD-SR

# 运行Exp2
python experiments/train.py \
    --experiment_name Exp2_Standard_KD \
    --model_name Qwen/Qwen2.5-1.5B \
    --teacher_name Qwen/Qwen2.5-7B \
    --teacher_load_in_4bit True \
    --train_data data/GeoSR-Train.json \
    --test_data data/GeoSR-Bench.json \
    --loss_fn standard_kd \
    --kd_alpha 0.5 \
    --temperature 2.0 \
    --learning_rate 2e-5 \
    --batch_size 16 \
    --num_epochs 3 \
    --seed 42 \
    --output_dir outputs/Exp2_Standard_KD \
    --logging_steps 50 \
    --eval_steps 500
```

**预期结果**:
- 准确率: ~70-75% (相比Exp1提升5-10%)
- 训练时间: ~2小时
- 输出文件: `outputs/Exp2_Standard_KD/best_model/`

**验证检查点**:
- [ ] Exp2准确率 > Exp1准确率（验证蒸馏有效性）
- [ ] 训练稳定，无NaN损失
- [ ] 日志文件完整

## 三、阶段B：单组件消融（可并行）

> **前提条件**: Exp1和Exp2已完成且验证通过

### 3.1 Exp3a: C1-Uniform-Weight（均匀权重对照）

**目的**: 验证C1空间关系权重的必要性，对比均匀权重 vs 自适应权重

**配置**:
- 基于Exp2，添加C1组件
- 空间关系权重: [1.0, 1.0, 1.0, 1.0]（均匀）
- 关系类型: direction(方向), topological(拓扑), metric(度量), composite(组合)

**执行命令**:
```bash
cd D:/30_keyan/GeoKD-SR

python experiments/train.py \
    --experiment_name Exp3a_C1_Uniform_Weight \
    --model_name Qwen/Qwen2.5-1.5B \
    --teacher_name Qwen/Qwen2.5-7B \
    --teacher_load_in_4bit True \
    --train_data data/GeoSR-Train.json \
    --test_data data/GeoSR-Bench.json \
    --loss_fn spatial_relation_distill \
    --relation_weights 1.0 1.0 1.0 1.0 \
    --kd_alpha 0.5 \
    --temperature 2.0 \
    --learning_rate 2e-5 \
    --batch_size 16 \
    --num_epochs 3 \
    --seed 42 \
    --output_dir outputs/Exp3a_C1_Uniform_Weight \
    --logging_steps 50 \
    --eval_steps 500
```

**预期结果**:
- 准确率: ~71-76%
- 与Exp2对比：验证"加权"是否有效

---

### 3.2 Exp3b: C1-Adaptive-Weight（自适应权重）

**目的**: 验证基于关系类型的自适应权重策略

**配置**:

- 空间关系权重: [1.5, 1.3, 1.0, 1.8]
  - w(directional) = 1.5（方向关系较难，高权重）
  - w(topological) = 1.3（拓扑关系中等）
  - w(metric) = 1.0（度量关系相对简单）
  - w(composite) = 1.8（组合关系最复杂，最高权重）

**执行命令**:
```bash
cd D:/30_keyan/GeoKD-SR

    --experiment_name Exp3b_C1_Adaptive_Weight \
    --model_name Qwen/Qwen2.5-1.5B \
    --teacher_name Qwen/Qwen2.5-7B \
    --teacher_load_in_4bit True \
    --train_data data/GeoSR-Train.json \
    --test_data data/GeoSR-Bench.json \
    --loss_fn spatial_relation_distill \
    --relation_weights 1.5 1.3 1.0 1.8 \
    --kd_alpha 0.5 \
    --temperature 2.0 \
    --learning_rate 2e-5 \
    --batch_size 16 \
    --num_epochs 3 \
    --seed 42 \
    --output_dir outputs/Exp3b_C1_Adaptive_Weight \
    --logging_steps 50 \
    --eval_steps 500
```

**预期结果**:
- 准确率: ~72-77%
- 与Exp3a对比：验证"自适应权重"优于"均匀权重"
