# 1.3 输入输出规范

## 概述

本文档定义GeoKD-SR数据集的输入输出格式规范，包括数据文件格式、字段定义、数据分布要求等。

---

## 一、输入数据格式

### 1.1 文件格式

训练数据采用**JSONL格式**（每行一个JSON对象）：

```bash
data/
├── train.jsonl      # 训练集
├── validation.jsonl # 验证集
└── test.jsonl       # 测试集
```

每行JSON对象必须包含以下字段：

## 二、必需字段定义

### 2.1 基础字段 (V2.0)

| 字段名 | 类型 | 必需 | 说明 | 示例 |
|--------|------|------|------|------|
| id | string | ✓ | 格式: `geosr_{序号}` | "geosr_001" |
| spatial_relation_type | enum | ✓ | 取值: `directional`, `topological`, `metric`, `composite` | "directional" |
| topology_subtype | string | 条件 | 拓扑子类型 (topological时必需) | "within" |
| question | string | ✓ | 问题文本，中文 | "北京在上海的什么方向？" |
| answer | string | ✓ | 答案文本，中文 | "西北方向" |
| reasoning_chain | array | ✓ | **5步推理链** | 见完整结构 |
| entities | array | ✓ | 至少2个实体，每实体含name/type/coords | [...] |
| spatial_tokens | array | ✓ | 空间相关关键词列表 | ["北京", "上海"] |
| entity_to_token | object | ✓ | 实体到Token索引映射 | {...} |
| difficulty | enum | ✓ | 取值: `easy`, `medium`, `hard` | "easy" |
| difficulty_score | float | ✓ | 难度评分，1.0-5.0 | 1.5 |

### 2.2 spatial_relation_type 枚举值 (V2.0更新)

```yaml
spatial_relation_types:
  - directional    # 方向关系 (如: 北京在上海的什么方向)
  - topological    # 拓扑关系 (如: 故宫是否在北京市内)
  - metric         # 度量关系 (如: 北京到上海的距离)
  - composite      # 组合关系 (如: 从北京向南500公里的城市)
```

### 2.3 topology_subtype 枚举值 (V2.0新增)

```yaml
topology_subtypes:
  - within      # A在B内部 (如: 故宫在北京市内)
  - contains    # B包含A (如: 北京市包含故宫)
  - adjacent    # A与B相邻 (如: 河北省与山东省相邻)
  - disjoint    # A与B分离 (如: 海南省与黑龙江省不接壤)
  - overlap     # A与B交叉 (如: 长江流经多个省份)
```

### 2.4 coordinates 字段格式 (V2.0更新)

```typescript
interface Coordinates {
  [entity_name: string]: {
    lat: number;  // 纬度, 范围: 3.86-53.55 (中国境内)
    lon: number;  // 经度, 范围: 73.66-135.05 (中国境内)
  }
}

// 或使用数组格式 (推荐):
// coords: [经度, 纬度]  // 注意：经度在前
```

**约束条件：**
- 纬度精度: 至少4位小数
- 经度精度: 至少4位小数
- WGS84坐标系
- 必须在中国境内范围内

### 2.5 entity_to_token 字段格式 (V2.0更新)

```typescript
interface EntityToToken {
  [entity_name: string]: {
    char_start: number;    // 字符起始位置
    char_end: number;      // 字符结束位置
    token_indices: number[]; // Token索引列表
  }
}
```

### 2.6 reasoning_chain 字段格式

每个推理步骤必须包含：

| 字段名 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| step | integer | ✓ | 步骤编号 (1-5) |
| name | string | ✓ | 步骤名称 |
| action | string | ✓ | 动作类型 |
| content | string | ✓ | 步骤描述 |
| [step_specific] | any | 条件 | 步骤特定字段 |

**5步推理链标准结构：**

1. **entity_identification** - 实体识别
2. **spatial_relation_extraction** - 空间关系提取
3. **coordinate_retrieval** - 坐标获取与Token映射
4. **spatial_calculation** - 空间计算
5. **answer_generation** - 答案生成

### 4.4 单实体频率限制

单一实体在数据集中出现的频率不应超过5%：

```python
def check_entity_frequency(dataset, max_freq=0.05):
    """检查实体频率"""
    from collections import Counter

    entity_counter = Counter()
    total = len(dataset)

    for item in dataset:
        for entity in item.get("entities", []):
            entity_counter[entity["name"]] += 1

    # 检查是否有实体超过频率限制
    overrepresented = []
    for entity, count in entity_counter.items():
        freq = count / total
        if freq > max_freq:
            overrepresented.append((entity, freq))

    return overrepresented

# 使用示例
overrepresented = check_entity_frequency(train_data)
if overrepresented:
    print(f"警告：以下实体出现频率过高：{overrepresented}")
```

```python
from sklearn.model_selection import train_test_split
import pandas as pd

def stratified_split(dataset):
    """分层抽样划分数据集"""
    df = pd.DataFrame(dataset)

    # 按spatial_relation_type和difficulty分层
    df['stratify_key'] = df['spatial_relation_type'] + '_' + df['difficulty']

    train_val, test = train_test_split(
        df,
        test_size=0.15,
        stratify=df['stratify_key'],
        random_state=42
    )

    train, val = train_test_split(
        train_val,
        test_size=0.176,  # 0.15 / 0.85 ≈ 0.176
        stratify=train_val['stratify_key'],
        random_state=42
    )

    return train, val, test
```

## 七、数据质量检查脚本 (V2.0更新)

### 7.1 完整检查脚本

```python
#!/usr/bin/env python3
"""数据质量检查脚本 V2.0"""

import json
import sys
from pathlib import Path
from collections import Counter
import numpy as np

def check_jsonl_format(file_path):
    """检查JSONL格式"""
    try:
        with open(file_path, 'r', encoding='utf-8') as f:
            for line_num, line in enumerate(f, 1):
                if line.strip():
                    json.loads(line)
        return True, "JSONL格式正确"
    except json.JSONDecodeError as e:
        return False, f"第{line_num}行JSON解析错误: {e}"

def check_required_fields(file_path):
    """检查必需字段 V2.0"""
    required_fields = [
        'id', 'spatial_relation_type', 'question', 'answer',
        'reasoning_chain', 'entities', 'spatial_tokens',
        'entity_to_token', 'difficulty', 'difficulty_score'
    ]

    missing_fields = []

    with open(file_path, 'r', encoding='utf-8') as f:
        for line_num, line in enumerate(f, 1):
            if not line.strip():
                continue
            data = json.loads(line)

            for field in required_fields:
                if field not in data:
                    missing_fields.append(f"第{line_num}行: 缺少字段 '{field}'")

            # 检查topological类型是否有topology_subtype
            if data.get('spatial_relation_type') == 'topological':
                if 'topology_subtype' not in data or data['topology_subtype'] is None:
                    missing_fields.append(f"第{line_num}行: topological类型缺少topology_subtype")

    if missing_fields:
        return False, "; ".join(missing_fields[:10])
    return True, "必需字段完整"

def check_relation_distribution(file_path):
    """检查关系类型分布 V2.0"""
    relation_counts = Counter()

    with open(file_path, 'r', encoding='utf-8') as f:
        for line in f:
            if line.strip():
                data = json.loads(line)
                relation_counts[data.get('spatial_relation_type', 'unknown')] += 1

    if len(relation_counts) < 2:
        return False, f"关系类型过少: {list(relation_counts.keys())}"

    # 检查分布是否符合要求
    total = sum(relation_counts.values())
    expected_ratios = {
        'directional': 0.25,
        'topological': 0.275,
        'metric': 0.275,
        'composite': 0.20
    }

    deviations = []
    for rel_type, expected_ratio in expected_ratios.items():
        actual_ratio = relation_counts.get(rel_type, 0) / total
        deviation = abs(actual_ratio - expected_ratio)
        if deviation > 0.05:  # 允许5%偏差
            deviations.append(f"{rel_type}: 期望{expected_ratio:.1%}, 实际{actual_ratio:.1%}")

    if deviations:
        return False, f"分布偏差过大: {deviations}"

    cv = np.std(list(relation_counts.values())) / np.mean(list(relation_counts.values()))

    if cv >= 0.2:
        return False, f"分布不均衡，CV={cv:.3f}，建议CV<0.2"

    return True, f"分布均衡，CV={cv:.3f}"

def check_difficulty_distribution(file_path):
    """检查难度分布 V2.0新增"""
    difficulty_counts = Counter()

    with open(file_path, 'r', encoding='utf-8') as f:
        for line in f:
            if line.strip():
                data = json.loads(line)
                difficulty_counts[data.get('difficulty', 'unknown')] += 1

    total = sum(difficulty_counts.values())
    expected_ratios = {
        'easy': 0.30,
        'medium': 0.50,
        'hard': 0.20
    }

    deviations = []
    for diff, expected_ratio in expected_ratios.items():
        actual_ratio = difficulty_counts.get(diff, 0) / total
        deviation = abs(actual_ratio - expected_ratio)
        if deviation > 0.05:
            deviations.append(f"{diff}: 期望{expected_ratio:.1%}, 实际{actual_ratio:.1%}")

    if deviations:
        return False, f"难度分布偏差: {deviations}"

    return True, f"难度分布符合要求"

def check_entity_frequency(file_path, max_freq=0.05):
    """检查实体频率"""
    entity_counter = Counter()
    total = 0

    with open(file_path, 'r', encoding='utf-8') as f:
        for line in f:
            if line.strip():
                data = json.loads(line)
                total += 1
                for entity in data.get('entities', []):
                    if isinstance(entity, dict):
                        entity_counter[entity.get('name', '')] += 1
                    else:
                        entity_counter[entity] += 1

    overrepresented = []
    for entity, count in entity_counter.items():
        freq = count / total
        if freq > max_freq:
            overrepresented.append((entity, freq))

    if overrepresented:
        return False, f"频率过高实体: {[(e, f'{f:.2%}') for e, f in overrepresented[:5]]}"
    return True, "实体频率符合要求"

def check_difficulty_score_range(file_path):
    """检查difficulty_score范围 V2.0新增"""
    invalid_scores = []

    with open(file_path, 'r', encoding='utf-8') as f:
        for line_num, line in enumerate(f, 1):
            if not line.strip():
                continue
            data = json.loads(line)
            score = data.get('difficulty_score', 0)

            if not (1.0 <= score <= 5.0):
                invalid_scores.append(f"第{line_num}行: difficulty_score={score} 不在[1.0, 5.0]范围内")

    if invalid_scores:
        return False, "; ".join(invalid_scores[:5])
    return True, "difficulty_score范围正确"

def main():
    if len(sys.argv) < 2:
        print("用法: python check_data_quality.py <data_file.jsonl>")
        sys.exit(1)

    file_path = sys.argv[1]

    checks = [
        ("JSONL格式", check_jsonl_format),
        ("必需字段", check_required_fields),
        ("关系分布", check_relation_distribution),
        ("难度分布", check_difficulty_distribution),
        ("实体频率", check_entity_frequency),
        ("难度评分范围", check_difficulty_score_range)
    ]

    print(f"\n检查文件: {file_path}")
    print("=" * 60)

    all_passed = True
    for name, check_func in checks:
        passed, message = check_func(file_path)
        status = "✓ PASS" if passed else "✗ FAIL"
        print(f"{status:8} {name}: {message}")
        if not passed:
            all_passed = False

    print("=" * 60)
    if all_passed:
        print("\n所有检查通过！")
        sys.exit(0)
    else:
        print("\n部分检查失败，请修复后重试。")
        sys.exit(1)

if __name__ == "__main__":
    main()
```

## 八、数据导入/导出API (V2.0更新)

### 8.1 Python API示例

```python
class GeoReasoningDataset:
    """地理空间推理数据集类 V2.0"""

    def __init__(self, file_path):
        self.file_path = file_path
        self.data = []

    def load(self):
        """加载数据"""
        with open(self.file_path, 'r', encoding='utf-8') as f:
            for line in f:
                if line.strip():
                    self.data.append(json.loads(line))
        return self

    def save(self, output_path):
        """保存数据"""
        with open(output_path, 'w', encoding='utf-8') as f:
            for item in self.data:
                f.write(json.dumps(item, ensure_ascii=False) + '\n')

    def filter_by_type(self, spatial_relation_type):
        """按空间关系类型过滤"""
        return [d for d in self.data if d['spatial_relation_type'] == spatial_relation_type]

    def filter_by_difficulty(self, difficulty):
        """按难度过滤"""
        return [d for d in self.data if d['difficulty'] == difficulty]

    def get_statistics(self):
        """获取数据统计信息"""
        return {
            'total': len(self.data),
            'relation_types': Counter(d['spatial_relation_type'] for d in self.data),
            'difficulties': Counter(d['difficulty'] for d in self.data),
            'entities': Counter(e['name'] for d in self.data for e in d.get('entities', [])),
            'avg_difficulty_score': np.mean([d['difficulty_score'] for d in self.data])
        }

# 使用示例
dataset = GeoReasoningDataset('data/train.jsonl').load()
stats = dataset.get_statistics()
print(f"总数: {stats['total']}")
print(f"关系类型分布: {stats['relation_types']}")
print(f"难度分布: {stats['difficulties']}")
print(f"平均难度评分: {stats['avg_difficulty_score']:.2f}")
```
