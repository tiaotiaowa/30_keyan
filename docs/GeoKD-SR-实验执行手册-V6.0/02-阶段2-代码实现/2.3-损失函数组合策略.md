## 一、HybridKLDistillationLoss 设计

### 1.1 问题背景

**D3-8问题**: 当前损失函数组合策略过于简单，缺乏理论依据。

**解决方案**: 设计混合KL蒸馏损失，结合Forward KL和Reverse KL的优势，并使用动态权重调整。

### 1.2 核心设计

#### 1.2.1 Forward KL vs Reverse KL

| 类型 | 公式 | 优势 | 劣势 |
|------|------|------|------|
| **Forward KL** | KL(P\|\|Q) = Σ p(x) log(p(x)/q(x)) | 鼓励学生覆盖教师所有支持 | 可能产生过度平滑 |
| **Reverse KL** | KL(Q\|\|P) = Σ q(x) log(q(x)/p(x)) | 鼓励学生集中在教师高概率区 | 可能丢失长尾知识 |

**混合策略**: 早期使用更多Forward KL探索，后期使用更多Reverse KL精炼。

#### 1.2.2 动态权重公式

```
α(t) = α_init - (α_init - α_final) × (t / T)^β

其中:
- t: 当前epoch
- T: 总epoch数
- α_init: 初始Forward KL权重 (默认0.8)
- α_final: 最终Forward KL权重 (默认0.2)
- β: 衰减指数 (默认1.0，线性衰减)
```

#### 1.2.3 温度参数策略

```
T(t) = T_init - (T_init - T_final) × (t / T)

其中:
- T_init = 4.0 (早期，软化分布)
- T_final = 1.0 (后期，接近真实分布)
```

### 1.3 完整实现

```python
# models/losses/hybrid_kl_loss.py

import torch
import torch.nn as nn
import torch.nn.functional as F
from typing import Dict, Optional, Union
import math

class HybridKLDistillationLoss(nn.Module):
    """
    混合KL蒸馏损失

    解决D3-8: 损失函数组合策略缺乏理论依据

    设计思路:
    1. 结合Forward KL (探索) 和 Reverse KL (利用)
    2. 使用动态权重调整训练策略
    3. 配合温度参数退火

    数学公式:
        L_total = α(t) × L_forward + (1-α(t)) × L_reverse + L_hard

        其中:
        - L_forward = T² × KL(P_teacher || P_student)
        - L_reverse = T² × KL(P_student || P_teacher)
        - L_hard = CE(P_student, y_true)
        - α(t) = α_init - (α_init - α_final) × (t/T)^β
    """

    def __init__(
        self,
        # KL权重配置
        alpha_init: float = 0.8,
        alpha_final: float = 0.2,
        beta: float = 1.0,
        # 温度配置
        temperature_init: float = 4.0,
        temperature_final: float = 1.0,
        # 损失权重
        soft_weight: float = 0.7,
        hard_weight: float = 0.3,
        # 训练配置
        total_epochs: int = 10,
        warmup_epochs: int = 1
    ):
        super().__init__()

        # 保存配置
        self.alpha_init = alpha_init
        self.alpha_final = alpha_final
        self.beta = beta
        self.temp_init = temperature_init
        self.temp_final = temperature_final
        self.soft_weight = soft_weight
        self.hard_weight = hard_weight
        self.total_epochs = total_epochs
        self.warmup_epochs = warmup_epochs

        # 当前训练状态
        self.current_epoch = 0
        self.global_step = 0

        # 统计信息
        self.register_buffer("loss_history", torch.zeros(total_epochs, 4))
        # 0: forward_kl, 1: reverse_kl, 2: soft_loss, 3: hard_loss

    def set_epoch(self, epoch: int):
        """设置当前训练epoch"""
        self.current_epoch = epoch
        if epoch >= self.total_epochs:
            self.total_epochs = epoch + 1
            self.loss_history = torch.zeros(epoch + 1, 4)

    def step(self):
        """增加全局步数"""
        self.global_step += 1

    def get_alpha(self, epoch: Optional[int] = None) -> float:
        """
        获取当前epoch的Forward KL权重

        Args:
            epoch: 指定epoch，默认使用self.current_epoch

        Returns:
            alpha值 ∈ [0, 1]
        """
        if epoch is None:
            epoch = self.current_epoch

        # Warmup期间使用固定权重
        if epoch < self.warmup_epochs:
            return self.alpha_init

        # 计算进度
        progress = (epoch - self.warmup_epochs) / (self.total_epochs - self.warmup_epochs)
        progress = min(progress, 1.0)

        # 计算衰减权重
        alpha = self.alpha_init - (self.alpha_init - self.alpha_final) * (progress ** self.beta)

        return alpha

    def get_temperature(self, epoch: Optional[int] = None) -> float:
        """
        获取当前epoch的温度参数

        Returns:
            温度值 ∈ [temp_final, temp_init]
        """
        if epoch is None:
            epoch = self.current_epoch

        # Warmup期间使用初始温度
        if epoch < self.warmup_epochs:
            return self.temp_init

        # 线性退火
        progress = (epoch - self.warmup_epochs) / (self.total_epochs - self.warmup_epochs)
        progress = min(progress, 1.0)

        temperature = self.temp_init - (self.temp_init - self.temp_final) * progress

        return temperature

    def compute_forward_kl(
        self,
        log_student: torch.Tensor,
        log_teacher: torch.Tensor
    ) -> torch.Tensor:
        """
        计算Forward KL散度: KL(P_teacher || P_student)

        Args:
            log_student: log P_student [B, S, V]
            log_teacher: log P_teacher [B, S, V]

        Returns:
            KL散度值 (标量)
        """
        # Forward KL: KL(Q||P) = Σ q × (log q - log p)
        # 其中 Q = teacher, P = student
        kl = (torch.exp(log_teacher) * (log_teacher - log_student)).sum(dim=-1)
        return kl.mean()

    def compute_reverse_kl(
        self,
        log_student: torch.Tensor,
        log_teacher: torch.Tensor
    ) -> torch.Tensor:
        """
        计算Reverse KL散度: KL(P_student || P_teacher)

        Args:
            log_student: log P_student [B, S, V]
            log_teacher: log P_teacher [B, S, V]

        Returns:
            KL散度值 (标量)
        """
        # Reverse KL: KL(P||Q) = Σ p × (log p - log q)
        # 其中 P = student, Q = teacher
        kl = (torch.exp(log_student) * (log_student - log_teacher)).sum(dim=-1)
        return kl.mean()

    def forward(
        self,
        student_logits: torch.Tensor,
        teacher_logits: torch.Tensor,
        labels: torch.Tensor,
        attention_mask: Optional[torch.Tensor] = None,
        return_details: bool = False
    ) -> Union[torch.Tensor, Dict[str, torch.Tensor]]:
        """
        计算混合KL蒸馏损失

        Args:
            student_logits: 学生模型输出 [B, S, V]
            teacher_logits: 教师模型输出 [B, S, V]
            labels: 真实标签 [B, S]
            attention_mask: 注意力掩码 [B, S]
            return_details: 是否返回损失详情

        Returns:
            总损失，或损失字典
        """
        # 获取当前配置
        alpha = self.get_alpha()
        temperature = self.get_temperature()

        # 计算log概率
        log_student = F.log_softmax(
            student_logits / temperature,
            dim=-1
        )
        log_teacher = F.log_softmax(
            teacher_logits / temperature,
            dim=-1
        )

        # 计算KL散度
        forward_kl = self.compute_forward_kl(log_student, log_teacher)
        reverse_kl = self.compute_reverse_kl(log_student, log_teacher)

        # 混合软标签损失
        soft_loss = (alpha * forward_kl + (1 - alpha) * reverse_kl)
        soft_loss = soft_loss * (temperature ** 2)

        # 硬标签损失
        hard_loss = F.cross_entropy(
            student_logits.view(-1, student_logits.size(-1)),
            labels.view(-1),
            ignore_index=-100
        )

        # 组合
        total_loss = self.soft_weight * soft_loss + self.hard_weight * hard_loss

        # 记录统计
        if self.current_epoch < len(self.loss_history):
            self.loss_history[self.current_epoch] = torch.tensor([
                forward_kl.item(),
                reverse_kl.item(),
                soft_loss.item(),
                hard_loss.item()
            ])

        self.step()

        if return_details:
            return {
                "loss": total_loss,
                "forward_kl": forward_kl,
                "reverse_kl": reverse_kl,
                "soft_loss": soft_loss,
                "hard_loss": hard_loss,
                "alpha": alpha,
                "temperature": temperature
            }

        return total_loss

    def get_statistics(self) -> Dict[str, float]:
        """获取训练统计信息"""
        if self.current_epoch == 0:
            return {}

        history = self.loss_history[:self.current_epoch + 1]

        return {
            "avg_forward_kl": history[:, 0].mean().item(),
            "avg_reverse_kl": history[:, 1].mean().item(),
            "avg_soft_loss": history[:, 2].mean().item(),
            "avg_hard_loss": history[:, 3].mean().item(),
            "current_alpha": self.get_alpha(),
            "current_temperature": self.get_temperature()
        }


class AdaptiveHybridKLLoss(HybridKLDistillationLoss):
    """
    自适应混合KL损失

    根据训练进度自动调整权重
    """

    def __init__(
        self,
        *args,
        adapt_strategy: str = "linear",
        **kwargs
    ):
        super().__init__(*args, **kwargs)
        self.adapt_strategy = adapt_strategy

    def get_alpha(self, epoch: Optional[int] = None) -> float:
        """自适应获取alpha权重"""
        if epoch is None:
            epoch = self.current_epoch

        if epoch < self.warmup_epochs:
            return self.alpha_init

        progress = (epoch - self.warmup_epochs) / (self.total_epochs - self.warmup_epochs)

        if self.adapt_strategy == "linear":
            alpha = self.alpha_init - (self.alpha_init - self.alpha_final) * progress

        elif self.adapt_strategy == "cosine":
            # 余弦退火
            alpha = self.alpha_final + 0.5 * (self.alpha_init - self.alpha_final) * (
                1 + math.cos(math.pi * progress)
            )

        elif self.adapt_strategy == "exponential":
            # 指数衰减
            alpha = self.alpha_final + (self.alpha_init - self.alpha_final) * (
                0.5 ** progress
            )

        else:
            raise ValueError(f"Unknown adapt_strategy: {self.adapt_strategy}")

        return alpha
```

## 二、ProgressiveDataScheduler 设计

### 2.1 问题背景

**D3-9问题**: 缺乏数据调度策略，未体现渐进式训练理念。

**解决方案**: 设计渐进式数据调度器，分阶段使用不同复杂度的数据。

### 2.2 核心设计

#### 2.2.1 三阶段训练策略

| 阶段 | 数据类型 | 比例 | 目标 |
|------|----------|------|------|
| **Stage 1** | 简单方向/拓扑关系 | 40% | 学习基础空间概念 |
| **Stage 2** | 度量/组合关系 | 40% | 学习复杂推理 |
| **Stage 3** | 全部数据 + 困难样本 | 20% | 精炼和泛化 |

#### 2.2.2 数据复杂度度量

```python
# 复杂度评分标准
def compute_complexity(sample: Dict) -> float:
    """
    计算样本复杂度分数

    因子:
    1. 关系类型: directional(1) < topological(2) < metric(3) < hybrid(4)
    2. 问题长度: 越长越复杂
    3. 选项数量: 越多越复杂
    """
    type_score = {
        "directional": 1.0,
        "topological": 2.0,
        "metric": 2.5,
        "hybrid": 4.0
    }[sample["relation_type"]]

    length_score = min(len(sample["question"]) / 200, 1.0)
    option_score = min(sample.get("num_options", 4) / 8, 1.0)

    return 0.5 * type_score + 0.3 * length_score + 0.2 * option_score
```

```python
# models/data/progressive_scheduler.py

import torch
from torch.utils.data import Dataset, DataLoader, Sampler
from typing import Dict, List, Optional, Tuple
import numpy as np
from collections import defaultdict

class ProgressiveDataScheduler:
    """
    渐进式数据调度器

    解决D3-9: 数据调度策略缺失

    功能:
    1. 按复杂度分层数据
    2. 分阶段调度不同复杂度的数据
    3. 支持动态调整批次组成
    """

    def __init__(
        self,
        dataset: Dataset,
        # 阶段配置
        num_stages: int = 3,
        stage_ratios: List[float] = None,  # 每阶段数据比例
        # 复杂度配置
        complexity_bins: int = 5,
        # 调度策略
        schedule_type: str = "progressive",  # "progressive", "mixed", "curriculum"
        # 采样配置
        shuffle: bool = True,
        seed: int = 42
    ):
        self.dataset = dataset
        self.num_stages = num_stages
        self.stage_ratios = stage_ratios or [0.4, 0.4, 0.2]
        self.complexity_bins = complexity_bins
        self.schedule_type = schedule_type
        self.shuffle = shuffle
        self.seed = seed

        # 初始化
        self.rng = np.random.RandomState(seed)
        self._initialize_data()

    def _initialize_data(self):
        """初始化数据分层"""
        # 计算每个样本的复杂度
        self.complexities = []
        self.data_by_relation = defaultdict(list)

        for idx in range(len(self.dataset)):
            item = self.dataset[idx]
            complexity = self._compute_complexity(item)
            self.complexities.append(complexity)

            relation_type = item.get("relation_type", "unknown")
            self.data_by_relation[relation_type].append((idx, complexity))

        # 转换为numpy数组
        self.complexities = np.array(self.complexities)

        # 分层到bin
        self.bins = self._create_bins()

        # 初始化阶段数据索引
        self._create_stage_data()

    def _compute_complexity(self, item: Dict) -> float:
        """计算单个样本的复杂度"""
        # 关系类型分数
        type_score = {
            "directional": 1.0,
            "topological": 2.0,
            "metric": 2.5,
            "hybrid": 4.0,
            "unknown": 2.0
        }.get(item.get("relation_type", "unknown"), 2.0)

        # 长度分数 (归一化到[0,1])
        question = item.get("question", "")
        length_score = min(len(question) / 200, 1.0)

        # 组合
        complexity = 0.6 * type_score + 0.4 * length_score

        return complexity

    def _create_bins(self) -> List[Tuple[float, float]]:
        """创建复杂度分层边界"""
        min_comp = self.complexities.min()
        max_comp = self.complexities.max()

        bins = []
        for i in range(self.complexity_bins):
            start = min_comp + (max_comp - min_comp) * i / self.complexity_bins
            end = min_comp + (max_comp - min_comp) * (i + 1) / self.complexity_bins
            bins.append((start, end))

        return bins

    def _create_stage_data(self):
        """为每个阶段创建数据索引"""
        self.stage_indices = []

        for stage in range(self.num_stages):
            # 计算当前阶段应该使用的复杂度范围
            bin_start = int(stage * self.complexity_bins / self.num_stages)
            bin_end = int((stage + 1) * self.complexity_bins / self.num_stages)

            # 获取该复杂度范围的样本
            stage_idx = []
            for bin_idx in range(bin_start, min(bin_end, self.complexity_bins)):
                bin_start_val, bin_end_val = self.bins[bin_idx]
                mask = (self.complexities >= bin_start_val) & (
                    self.complexities < bin_end_val
                )
                stage_idx.extend(np.where(mask)[0].tolist())

            self.stage_indices.append(stage_idx)

    def get_stage_loader(
        self,
        stage: int,
        batch_size: int,
        num_workers: int = 0
    ) -> DataLoader:
        """获取指定阶段的DataLoader"""
        if stage >= len(self.stage_indices):
            raise ValueError(f"Stage {stage} out of range")

        indices = self.stage_indices[stage]

        if self.shuffle:
            self.rng.shuffle(indices)

        sampler = SubsetSampler(indices)

        return DataLoader(
            self.dataset,
            batch_size=batch_size,
            sampler=sampler,
            num_workers=num_workers,
            pin_memory=True
        )

    def get_mixed_loader(
        self,
        stage: int,
        batch_size: int,
        num_workers: int = 0,
        mix_ratio: float = 0.3
    ) -> DataLoader:
        """
        获取混合数据的DataLoader

        当前阶段数据 + 历史阶段数据 (用于防止遗忘)
        """
        # 当前阶段数据
        current_indices = self.stage_indices[stage]

        # 历史阶段数据
        history_indices = []
        for s in range(stage):
            history_indices.extend(self.stage_indices[s])

        # 按比例混合
        n_history = int(len(current_indices) * mix_ratio)
        if history_indices:
            history_sample = self.rng.choice(
                history_indices,
                min(n_history, len(history_indices)),
                replace=False
            ).tolist()
        else:
            history_sample = []

        mixed_indices = current_indices + history_sample

        if self.shuffle:
            self.rng.shuffle(mixed_indices)

        sampler = SubsetSampler(mixed_indices)

        return DataLoader(
            self.dataset,
            batch_size=batch_size,
            sampler=sampler,
            num_workers=num_workers,
            pin_memory=True
        )

    def get_statistics(self) -> Dict:
        """获取数据统计信息"""
        stats = {
            "total_samples": len(self.dataset),
            "complexity_range": (self.complexities.min(), self.complexities.max()),
            "complexity_mean": self.complexities.mean(),
            "samples_by_stage": [len(idx) for idx in self.stage_indices],
            "samples_by_relation": {
                k: len(v) for k, v in self.data_by_relation.items()
            }
        }
        return stats


class SubsetSampler(Sampler):
    """子集采样器"""

    def __init__(self, indices: List[int]):
        self.indices = indices

    def __iter__(self):
        return iter(self.indices)

    def __len__(self):
        return len(self.indices)


class CurriculumScheduler:
    """
    课程学习调度器

    根据训练进度动态调整数据难度
    """

    def __init__(
        self,
        scheduler: ProgressiveDataScheduler,
        total_epochs: int,
        stage_epochs: Optional[List[int]] = None
    ):
        self.scheduler = scheduler
        self.total_epochs = total_epochs

        # 每阶段训练epoch数
        if stage_epochs is None:
            self.stage_epochs = [
                total_epochs // 3,
                total_epochs // 3,
                total_epochs - 2 * (total_epochs // 3)
            ]
        else:
            self.stage_epochs = stage_epochs

        # 计算阶段边界
        self.stage_boundaries = []
        acc = 0
        for e in self.stage_epochs:
            acc += e
            self.stage_boundaries.append(acc)

    def get_stage(self, epoch: int) -> int:
        """获取当前epoch对应的阶段"""
        for stage, boundary in enumerate(self.stage_boundaries):
            if epoch < boundary:
                return stage
        return len(self.stage_boundaries) - 1

    def get_loader(
        self,
        epoch: int,
        batch_size: int,
        num_workers: int = 0,
        use_mixed: bool = True
    ) -> DataLoader:
        """获取当前epoch的DataLoader"""
        stage = self.get_stage(epoch)

        if use_mixed and stage > 0:
            return self.scheduler.get_mixed_loader(
                stage, batch_size, num_workers
            )
        else:
            return self.scheduler.get_stage_loader(
                stage, batch_size, num_workers
            )

        # 打印统计
        stage = curriculum.get_stage(epoch)
        stats = loss_fn.get_statistics()
        print(f"Epoch {epoch} (Stage {stage}): {stats}")
```

## 三、完整损失组合公式

### 3.1 GeoKD-SR完整损失

```
L_total = L_hard + L_soft + L_spatial + L_cot + L_attn

其中:
1. L_hard = CE(P_student, y_true)
2. L_soft = α(t) × KL(P_T||P_S) + (1-α(t)) × KL(P_S||P_T)
3. L_spatial = Σᵢ w_{rt(i)} × L_soft(x_i)
4. L_cot = λ × Σₗ MSE(h_S^l, h_T^l)
5. L_attn = μ × MSE(attn_S, attn_T)
```

### 3.2 权重配置表

| 损失项 | 权重符号 | 默认值 | 调整范围 | 说明 |
|--------|----------|--------|----------|------|
| 硬标签 | λ_hard | 0.3 | 0.2-0.5 | 保证基础性能 |
| 软标签 | λ_soft | 0.5 | 0.3-0.7 | 核心蒸馏 |
| 空间关系 | λ_spatial | 0.1 | 0.05-0.2 | C1组件 |
| 思维链 | λ_cot | 0.05 | 0.0-0.1 | C2组件 |
| 注意力 | λ_attn | 0.05 | 0.0-0.1 | C5组件 |
