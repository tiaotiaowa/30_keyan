## 二、环境配置步骤

### 2.1 创建虚拟环境

#### 使用Conda（推荐）

```bash
# 创建专用环境
conda create -n geokd-sr python=3.10 -y

# 激活环境
conda activate geokd-sr

# 设置pip镜像源（可选，加速下载）
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 使用venv

```bash
# 创建虚拟环境
python -m venv geokd-sr-env

# 激活环境
source geokd-sr-env/bin/activate  # Linux/Mac
# 或
geokd-sr-env\Scripts\activate     # Windows
```

### 2.2 安装PyTorch

#### CUDA 12.1版本

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

#### CUDA 11.8版本

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

## 五、环境验证

### 5.1 完整验证脚本

创建 `scripts/verify_env.py`：

```python
#!/usr/bin/env python3
"""环境验证脚本"""

import sys
import torch
import transformers
import peft
import accelerate
import datasets

def check_python_version():
    """检查Python版本"""
    version = sys.version_info
    if version.major >= 3 and version.minor >= 10:
        print(f"✓ Python版本: {version.major}.{version.minor}.{version.micro}")
        return True
    else:
        print(f"✗ Python版本过低: {version.major}.{version.minor}.{version.micro}")
        return False

def check_pytorch():
    """检查PyTorch和CUDA"""
    print(f"✓ PyTorch版本: {torch.__version__}")

    if torch.cuda.is_available():
        print(f"✓ CUDA可用: {torch.version.cuda}")
        print(f"✓ GPU数量: {torch.cuda.device_count()}")
        for i in range(torch.cuda.device_count()):
            props = torch.cuda.get_device_properties(i)
            print(f"  GPU {i}: {props.name} ({props.total_memory / 1024**3:.1f} GB)")
        return True
    else:
        print("✗ CUDA不可用")
        return False

def check_dependencies():
    """检查依赖包"""
    deps = {
        "transformers": transformers,
        "peft": peft,
        "accelerate": accelerate,
        "datasets": datasets
    }

    all_ok = True
    for name, module in deps.items():
        version = getattr(module, "__version__", "unknown")
        print(f"✓ {name}: {version}")

    return all_ok

def check_directories():
    """检查目录结构"""
    import os
    dirs = [
        "data/train", "data/test", "data/val",
        "models/teacher", "models/student",
        "outputs/checkpoints", "outputs/logs", "outputs/metrics",
        "scripts", "experiments"
    ]

    all_ok = True
    for dir_path in dirs:
        if os.path.exists(dir_path):
            print(f"✓ 目录存在: {dir_path}")
        else:
            print(f"✗ 目录缺失: {dir_path}")
            all_ok = False

    return all_ok

def main():
    """主函数"""
    print("="*50)
    print("GeoKD-SR 环境验证")
    print("="*50)

    results = {
        "Python版本": check_python_version(),
        "PyTorch/CUDA": check_pytorch(),
        "依赖包": check_dependencies(),
        "目录结构": check_directories()
    }

    print("="*50)
    print("验证结果汇总:")
    print("="*50)
    for name, passed in results.items():
        status = "✓ 通过" if passed else "✗ 失败"
        print(f"{name}: {status}")

    all_passed = all(results.values())
    if all_passed:
        print("\n✓ 所有检查通过，环境配置正确！")
        return 0
    else:
        print("\n✗ 部分检查失败，请修复上述问题")
        return 1

if __name__ == "__main__":
    sys.exit(main())
```

### 5.2 运行验证


## 六、常见问题排查

### 问题1：CUDA不可用

**现象**：`torch.cuda.is_available()` 返回 False

**排查步骤**：
1. 检查GPU驱动是否安装：`nvidia-smi`
2. 检查CUDA toolkit是否安装：`nvcc --version`
3. 检查PyTorch版本是否与CUDA匹配
4. 重新安装匹配的PyTorch版本

### 问题2：导入错误

**现象**：`ImportError: No module named 'xxx'`

**解决方案**：
```bash
pip install xxx
# 或
pip install -r requirements.txt
```

### 问题3：版本冲突

**现象**：依赖包版本不兼容

**解决方案**：
1. 使用conda解决依赖
2. 创建干净的新环境
3. 固定依赖版本

### 问题4：内存不足

**现象**：运行时OOM

**解决方案**：
1. 减小batch size
2. 使用gradient checkpointing
3. 使用混合精度训练
