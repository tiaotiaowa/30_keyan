#### 问题D3-02：GLM-5评估数据泄露风险仍未量化

**位置**: 7.3节 LLM辅助评测（第1425-1507行）

**问题描述**:

- 高风险场景依然存在：
  - 训练数据：GLM-5生成的GeoSR-Chain
  - 评估数据：GeoSR-Bench

- 没有提供n-gram重叠检测代码或检测计划

**影响分析**:
- 如果GLM-5见过类似地理问题，评估结果会严重虚高
- 投稿时审稿人会质疑：如何证明评估没有数据泄露？
- 这是学术诚信问题，可能导致拒稿

**改进建议**:
```python
# 必须添加数据泄露检测
def detect_data_leakage(train_data, test_data, eval_model):
    """
    检测训练数据和测试数据之间的泄露风险

    Returns:
        dict: 泄露风险指标
    """
    # 1. n-gram重叠检测
    from nltk import ngrams
    train_ngrams = set()
    for sample in train_data:
        tokens = sample['question'].split()
        train_ngrams.update(ngrams(tokens, n=4))
        train_ngrams.update(ngrams(tokens, n=5))

    overlap_counts = []
    for sample in test_data:
        tokens = sample['question'].split()
        test_ngrams = set(ngrams(tokens, n=4)) | set(ngrams(tokens, n=5))
        overlap = len(train_ngrams & test_ngrams)
        overlap_counts.append(overlap)

    # 2. 报告泄露率
    leakage_rate = sum(c > 0 for c in overlap_counts) / len(test_data)

    return {
        '4_gram_overlap_rate': leakage_rate,
        'avg_overlaps_per_sample': mean(overlap_counts),
        'high_risk_samples': [i for i, c in enumerate(overlap_counts) if c > 2]
    }

# 阈值：leakage_rate < 5% 可接受，>10% 需要更换评估方法
```

**影响分析**:

- 如果评估不稳定，所有基于GLM-5的结果都不可靠

**改进建议**:
```python
# 必须添加一致性验证
def validate_llm_evaluation_consistency(llm_evaluator, test_samples, human_labels=None):
    """
    验证LLM评估的一致性

    Returns:
        dict: 一致性指标
    """
    # 1. 内部一致性（稳定性）
    stability_scores = []
    for sample in test_samples[:100]:  # 采样100题
        scores = []
        for _ in range(3):  # 同一问题评估3次
            score = llm_evaluator.evaluate(sample)
            scores.append(score)
        stability_scores.append({
            'mean': mean(scores),
            'std': std(scores),
            'range': max(scores) - min(scores)
        })

    # 2. 目标：std < 5分（满分100分）
    unstable_samples = [s for s in stability_scores if s['std'] > 5]

    # 3. 人机一致性（如果有人类标注）
    if human_labels:
        correlation = pearson_correlation(llm_scores, human_scores)
        kappa = cohens_kappa(llm_categories, human_categories)
    else:
        correlation, kappa = None, None

    return {
        'internal_stability': {
            'avg_std': mean(s['std'] for s in stability_scores),
            'unstable_rate': len(unstable_samples) / len(stability_scores),
            'unstable_samples': unstable_samples[:10]  # 前10个不稳定样本
        },
        'human_agreement': {
            'pearson_correlation': correlation,
            'cohens_kappa': kappa
        },
        'acceptable': stability_scores and mean(s['std'] for s in stability_scores) < 5
    }

# 验证标准：
# - 内部稳定性：平均标准差 < 5分（满分100）
# - 人机一致性：Pearson相关系数 > 0.7, Cohen's Kappa > 0.6
# - 如果不满足，需要调整Prompt或使用其他评估方法
```

**优先级**: P1 - 强烈建议在实验开始前验证

**位置**: 6.1.3节 多重比较校正（第1237-1278行）

**问题描述**:
- 虽然设计了Holm-Bonferroni校正，但**应用于评测结果的部分不够完整**：
  - 评测指标的多次运行是否有统计显著性？
  - 不同模型之间的比较是否进行了统计检验？
  - 效应量计算是否包含所有主要对比？

**影响分析**:
- 无法判断性能提升是否真实显著
- 可能存在假阳性（看起来提升但实际不显著）

**改进建议**:
```python
# 必须添加完整的统计检验
def statistical_significance_test(results_dict):
    """
    对评测结果进行统计显著性检验

    Args:
        results_dict: {model_name: [run1_score, run2_score, ...]}

    Returns:
        dict: 统计检验结果
    """
    from scipy import stats
    import itertools

    # 1. 两两比较
    comparisons = []
    for model1, model2 in itertools.combinations(results_dict.keys(), 2):
        scores1 = results_dict[model1]
        scores2 = results_dict[model2]

        # t检验
        t_stat, p_value = stats.ttest_ind(scores1, scores2)

        # 效应量
        cohens_d = (mean(scores1) - mean(scores2)) / pooled_std(scores1, scores2)

        # Wilcoxon秩和检验（非参数）
        u_stat, p_value_wilcoxon = stats.mannwhitneyu(scores1, scores2)

        comparisons.append({
            'model1': model1,
            'model2': model2,
            'mean_diff': mean(scores1) - mean(scores2),
            'p_value': p_value,
            'p_value_wilcoxon': p_value_wilcoxon,
            'cohens_d': cohens_d,
            'significant': p_value < 0.05
        })

    # 2. Holm-Bonferroni校正
    p_values = [c['p_value'] for c in comparisons]
    rejected, adjusted_p = holm_bonferroni_correction(p_values, alpha=0.05)

    for i, comp in enumerate(comparisons):
        comp['adjusted_p'] = adjusted_p[i]
        comp['significant_after_correction'] = rejected[i]

    return {
        'comparisons': comparisons,
        'summary': {
            'total_comparisons': len(comparisons),
            'significant': sum(c['significant_after_correction'] for c in comparisons),
            'most_significant_improvement': max(comparisons, key=lambda x: x['cohens_d'])
        }
    }

# 报告格式：
# | 对比 | 均值差 | p值 | 校正p值 | Cohen's d | 显著性 |
# |------|--------|-----|---------|-----------|--------|
# | GeoKD-SR vs B2 | +5.2% | 0.003 | 0.018 | 0.85 | *** |
```
